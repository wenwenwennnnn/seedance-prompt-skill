# 景别与镜头术语表

## 景别术语

| 英文 | 中文 | 说明 |
|------|------|------|
| Extreme Wide Shot (EWS) | 大远景 | 展示广阔场景，人物极小 |
| Wide Shot (WS) | 远景 | 展示人物与环境关系 |
| Full Shot (FS) | 全景 | 展示人物全身 |
| Medium Shot (MS) | 中景 | 膝盖以上 |
| Medium Close-Up (MCU) | 中近景 | 腰部到头部 |
| Close-Up (CU) | 特写 | 肩部以上 |
| Extreme Close-Up (ECU) | 大特写 | 局部细节 |

## 镜头运动术语

| 英文 | 中文 | 说明 |
|------|------|------|
| static shot | 固定 | 无运动 |
| push in / dolly in | 推 | 向前推进 |
| pull out / dolly out | 拉 | 向后拉出 |
| pan left/right | 摇 | 水平旋转 |
| tilt up/down | 倾斜 | 垂直旋转 |
| follow shot / tracking shot | 跟 | 跟随主体移动 |
| crane up / elevating shot | 升 | 向上提升 |
| crane down / descending shot | 降 | 向下降低 |
| orbit / circling around | 环绕 | 绕主体旋转 |

## 竖屏构图关键词

9:16比例时添加：
```
vertical composition, centered or lower third composition, background layers
```
